# memAI-go

脳型記憶システムのGoライブラリ。扁桃体（感情検出）・短期記憶（作業記憶）・長期記憶（ベクトル検索+感情プライミング）を提供する。

## インストール

```bash
go get github.com/ieee0824/memAI-go
```

## 機能

- **感情検出** - キーワードベースの日本語感情検出（6分類）、LLM不要
- **短期記憶 (STM)** - 活性化減衰モデルの作業記憶、感情による減衰変調
- **長期記憶 (LTM)** - ベクトル検索＋多要素スコアリング＋感情プライミング
- **フィードバック検出** - ユーザーの反応から正負フィードバックを検出
- **ストレージ差し替え可能** - `MemoryStore` インターフェースで任意のバックエンドに対応
